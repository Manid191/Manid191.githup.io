# Universal Power Plant Feasibility

A comprehensive, web-based tool for evaluating the financial feasibility of power plant projects. This application supports various technologies (Biomass, Biogas, Solar, etc.) and allows for detailed modeling of technical, financial, and operational parameters.

## Features

*   **Dynamic Financial Modeling**: Calculate NPV, IRR, LCOE, DSCR, and Payback Period.
*   **Detailed Input Parameters**: Configure Capital Expenditures (CAPEX), Operational Expenditures (OPEX), Tax Holidays, Loan Terms, and more.
*   **Interactive Dashboard**: Visualize cash flows, revenue breakdown, and cost structures with dynamic charts.
*   **Report Generation**: Auto-generated financial reports with print-ready formatting.
*   **Data Persistence**: Your project data is automatically saved to your browser's local storage.
*   **Export Functionality**: Export your project data for backup or sharing.
*   **Resettable State**: Easily reset all data to default values for a fresh start.

## Getting Started

1.  **Open the Application**: Simply double-click `index.html` to run the application in your web browser. No server installation is required.
2.  **Configure Inputs**: Navigate to the "Parameters" tab to set up your project's capacity, costs, and financing.
3.  **View Financials**: Check the "Financials" tab for detailed annual cash flow tables.
4.  **Analyze Report**: The "Report" tab provides a summary suitable for printing or saving as PDF.
